How to run the DLS USAP chat app

Chat Server
1. Go to <your folder>\De La Salle USAP MP\De La Salle Usap MP\De La Salle USAP Server\dist\
2. Open the De_La_Salle_USAP_Server.jar file
3. Enter a Port Number (ex. 8000)
4. Click Start Server

Chat Client
1. Go to <your folder>\De La Salle USAP MP\De La Salle Usap MP\De La Salle USAP Client\dist
2. Open the De_La_Salle_USAP_Client.jar file
3. Enter a Username (max 15 characters)
4. Enter an IP Address (ex. 127.0.0.1)
5. Enther the Port Number that the Server is connected to
6. Click Login
